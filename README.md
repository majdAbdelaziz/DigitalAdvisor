# مستشارك الرقمي الآلي 🤖

أداة ذكاء اصطناعي عربية لكشف الاحتيال المالي الرقمي — الركيزة السادسة من مشروع الحماية الرقمية.

## GitHub Pages

1. أنشئ repository جديد
2. ارفع المحتوى كاملاً مع مجلدات css/ و js/ و blog/
3. Settings → Pages → main → Save

## الهيكل

```
/
├── index.html          ← الرئيسية + SEO كامل + Schema.org
├── tools.html          ← 3 أدوات تفاعلية
├── how-it-works.html   ← شرح التقنية
├── about.html          ← المصادر العلمية
├── sitemap.xml         ← SEO
├── robots.txt          ← SEO
├── css/style.css       ← Noto Kufi Arabic + نظام بنفسجي كامل
├── js/main.js          ← محرك كشف الاحتيال + counters
└── blog/               ← 6 مقالات + فهرس
```
